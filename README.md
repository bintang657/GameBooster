# Game Booster

A Flutter-powered Android Game Booster app with modern dark UI, neon gaming accents, real-time device stats (RAM, CPU temp, battery), and a BOOST button with loading animations.

## Features
- Dark mode with neon gaming theme
- Mock/real device stats (RAM usage, CPU temperature, battery)
- Large BOOST button with haptic feedback and animations
- Developed by Bintang

## Quick Start
1. Run `flutter pub get`
2. Run `flutter run` to build and install APK

## Permissions
- KILL_BACKGROUND_PROCESSES
- RESTART_PACKAGES
- BATTERY_STATS
